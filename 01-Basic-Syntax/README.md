## 📊 資料類型
9種基礎資料類型  

| 類型 | 分類 | 說明 | 範例 |
|------|------|------|------|
| int, float | 原生 | 數值類型 | `10`, `3.14` |
| str | 原生 | 字串類型 | `"hello"` |
| list | 原生 | 可變序列 | `[1, 2, 3]` |
| tuple | 原生 | 不可變序列 | `(1, 2, 3)` |
| dict | 原生 | 鍵值對映射 | `{"a": 1, "b": 2}` |
